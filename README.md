# online-food-delivery-system
This is a web-based application that allows users to browse food items,
place orders, and manage food delivery online.

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- Node.js
- Express.js
- MongoDB

## Features
- User login and registration
- Browse food items
- Add to cart
- Place orders
- Online payment

## How to Run
1. Clone the repository
2. Install dependencies using `npm install`
3. Create `.env` file
4. Run project using `npm start`
